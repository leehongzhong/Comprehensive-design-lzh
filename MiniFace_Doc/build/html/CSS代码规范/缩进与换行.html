7.1. 缩进与换行¶

7.1.1. 缩进¶

使用 4个空格做为一个缩进层级

示例：

1.selector {
2    margin: 0;
3    padding: 0;
4}

7.1.2. 换行¶

每行不得超过 120个字符，除非单行不可分割，对于超长的样式，在样式值的 空格处或 ‘,’ 后换行，建议按逻辑分组

解释：过长的代码不容易阅读与维护
