8.3. 注释与初始化¶

8.3.1. 单行注释¶

示例：

1// good
2// 初始化项目
3function init() {...}
4
5//bad
6/* 初始化项目 */
7function init() {...}
8
9function init() {...} //初始化项目

8.3.2. 多行注释¶

示例：

 1// good
 2/*
 3    初始化项目
 4    并获取数据
 5*/
 6function init() {...}
 7
 8//bad
 9// 初始化项目
10// 并获取数据
11function init() {...}

8.3.3. 初始化¶

使用 {} 初始化对象字面量

示例：

1// good
2let obj = {};
3
4// bad
5let obj = new Object();

使用 [] 初始化数组

1// good
2let obj = [];
3
4// bad
5let obj = new Array();
