8.4. 空格与空行¶

8.4.1. 空格¶

在大括号前放一个空格

示例；

1function test() {
2    console.log('test');
3}

在控制语句（if、while 等）的小括号前放一个空格

示例：

1if (isTrue) {
2    fight();
3}

函数调用及声明中，不在函数的参数列表前加空格

示例：

1function fight(形参1,形参2,...) {
2    console.log('Swooosh!');
3}
4
5fight(实参1,实参2,...);

除了自增/自减运算符外，使用空格把运算符隔开

示例：

1let x = y + 5;
2
3x++;
4
5y--;

8.4.2. 空行¶

以下几种情况需要空行：

  • 变量声明后（当变量声明在代码块的最后一行时，则无需空行）

  • 注释前（当注释在代码块的第一行时，则无需空行）

  • 代码块后（在函数调用、数组、对象中则无需空行）

  • 文件最后保留一个空行
