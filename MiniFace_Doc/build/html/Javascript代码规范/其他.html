8.6. 其他规范¶

8.6.1. 访问对象属性¶

统一使用 ‘[]’ 来访问对象属性

示例：

1let luke = {
2    jedi: true,
3    age: 28
4};
5
6let isJedi = luke['jedi'];

8.6.2. 字符串¶

使用单引号包裹字符串

示例：

1let name = 'Bob Parr';

8.6.3. 变量的声明¶

使用 let 声明每一个变量

示例：

1// good
2let goSportsTeam = true;
3
4// bad
5goSportsTeam = true;

8.6.4. else语句规范¶

把 else 放在 if 代码块关闭括号的同一行

示例：

1if (test) {
2    thing1();
3
4    thing2();
5} else {
6    thing3();
7}

8.6.5. 使用行尾逗号¶

示例；

 1// good
 2let story = [
 3    once,
 4    upon,
 5    aTime
 6];
 7
 8// bad
 9let story = [
10    once
11  , upon
12  , aTime
13];

8.6.6. 在每一个语句后使用分号¶

示例：

1// good
2let name = 'Skywalker';
3
4// bad
5let name = 'Skywalker'
