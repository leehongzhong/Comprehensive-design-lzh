8.1. 命名¶

8.1.1. 常规变量命名为驼峰式命名¶

示例：

 1// good
 2let cityText = '北京';
 3
 4let isNumber = false;
 5
 6let hasSelectCity = true;
 7
 8// bad
 9let citytext = '北京';
10
11let is_number = false;
12
13let has_selectCity = true;

8.1.2. 命名应具有特定含义¶

示例：

1// good
2let query;
3
4// bad
5let q;

8.1.3. Boolean 类型的变量或属性的命名¶

Boolean 类型的变量或属性命名以 is / has 开头

示例：

 1// good
 2let isSelected = false;
 3
 4let isShowCancel = false;
 5
 6let hasHotel = false;
 7
 8// bad
 9let selected = false;
10
11let showCancel = false;
12
13let hotel = false;

8.1.4. 私有属性的命名¶

使用下划线 _ 开头命名私有属性

示例：

1// good
2this._firstName = 'Panda';
3
4// bad
5this.__firstName__ = 'Panda';
6
7this.firstName_ = 'Panda';

8.1.5. 常量的命名¶

常量全大写，单词间用下划线连接

示例：

1const MAX_COUNT = 10;
