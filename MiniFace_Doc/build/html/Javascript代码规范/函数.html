8.5. 函数¶

8.5.1. 函数体¶

避免一个函数体过大，尽量拆分为功能清晰的单一函数

示例：

 1// good
 2function render() {
 3    renderList();
 4
 5    renderNotice();
 6};
 7
 8function renderList() {
 9    // 渲染列表
10};
11
12function renderNotice(){
13    // 渲染公告
14};
15
16// bad
17function render() {
18    // 渲染列表的代码
19    ...
20    // 渲染公告的代码
21    ...
22};

8.5.2. 函数参数¶

避免函数参数过多，可将多个参数用对象代替

示例：

1// good
2function showDialog(option) {
3    ...
4};
5
6// bad
7function showDialog(el, data, target, className, isShowClose) {
8    ...
9};

8.5.3. 自执行函数¶

自执行函数使用()来包裹，避免返回不必要的值

示例：

1(function test() {
2    ...
3})();
